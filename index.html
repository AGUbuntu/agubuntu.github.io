AGUbuntu

AGUbuntu is an unofficial Ubuntu flavour based on Lubuntu, created using CUBIC, for the old & low-spec lab computers at AGU

What exactly is AGUbuntu?

  • AGUbuntu is a Lubuntu-based distribution, specifically modified for computers and learners in Abdullah Gul University.
  • Our goal is to provide Linux the best & easy experience to the learners in Abdullah Gul University
  • Our distribution's source code is open source and is available at our GitHub
